Why Flexbox?

Flexbox is one dimensional layout method used to arrange items in rows or columns. The item will expand (thus flex) to adapt into containers.

Flexbox shines where other CSS features as float and position fail, like vertically centering a block of content inside its parent. This layout method will make your life easier, and with a lot less code.

Select the Flex

First thing we need to select the element that will become a flexible box. We will be using the value display on the parent of the element we are going to style.

For example:


                            section {
                                display: flex;
                            }

The <section> element is now a flex container and its children flex items.

Flex Direction

Flex items are laid out along two axis:

  • the main axis is the axis in which the flex-items are laid out in (in rows or columns). This axis starts at main start and ends at main end.
  • the cross axis is the axis perpendicular to the main axis. It starts at cross start and ends at cross end.

Using the property flex-direction you can specify the direction of the main axis (default is row). 


                                flex-direction: column;
                                flex-direction: row;
Flex Wrap

Having a fixed width or height in your layout can sometimes cause your flexbox children to overflow the container.

To fix this you can add a wrap declaration to your parent element, like this: 


                            section {
                                display: flex;
                                flex-wrap: wrap;
                            }

With this declaration, any overflow will be moved down to the next line. Easy Peasy!

Align and Justify

To align items along the two axis we use align-items and justify-content.

align-items controls the positioning of the flex items on the cross axis.

  • The default value is stretch, which stretches the items inside the parent along the cross axis.
  • To just center the item mantaining its intrinsic dimension, we can use the value center.
  • Other values like flex-start or flex-end, will align the items at the start or the end of the cross axis.

justify-content controls the positioning of the flex items on the main axis.

  • The default value is flex-start, which positions the items at the start of the main axis. Using flex-end will position them at the end of the axis.
  • To center the flex items, again, we can use the value center.
  • To distribute the items evenly along the main axis, we can use the value space-around. This will leave a bit of space at either end.
  • The value space-between is very similar to the previous one space-around, but it doesn't leave any space at either end.
